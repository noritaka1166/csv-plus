# CSV+

CSV+ はクロスプラットフォームで動作するフリーのCSVエディタです。

タブエディタなので、1つのウィンドウで一度に複数のファイルを開くことができます。

Windows, Linuxでの動作をサポートしています。

※現在このソフトは試験版です。
不具合やご要望は [m_enomoto@plus-one.tech](mailto:m_enomoto@plus-one.tech) までご連絡ください。


## 主な機能

- 複数のファイルをまとめて管理できるタブ機能
- セルの並び替え・検索
- 選択したセルの合計値を計算する集計機能
- 文字コード・改行コードの変換
